# pstatistics
A Blueprint extension witch tells your statistics over all servers. It tells you RAM, Storage, CPU Usage and much more.

## Installation



## Install https://blueprint.zip
<img width="248" height="58" alt="385768486-9e50c580-8168-4c14-a7a6-4f426edeac45" src="https://github.com/user-attachments/assets/ff7b8484-36a3-4636-8b50-b18c28384820" />



download file in [releases](https://github.com/cryxfncs/pstatistics/releases)

drag the file in /var/www/pterodactyl

***Bash*** `cd /var/www/pterodactyl`

***Bash*** `blueprint -install pstatistics`


If an error occurs open a ticket in our discord https://discord.gg/mQUXCEDB32
