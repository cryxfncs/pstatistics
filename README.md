# pstatistics
A Blueprint extension witch tells your statistics over all servers. It tells you RAM, Storage, CPU Usage and much more.

# Download on https://www.sourcexchange.net/products/pstatistics

download file in sourcexchange

drag the file in /var/www/pterodactyl

***Bash*** `cd /var/www/pterodactyl`

***Bash*** `blueprint -install pstatistics`
